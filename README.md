# Classifieds Marketplace Web Application

A full-stack web application that allows users to register, log in, post advertisements, browse products, and manage their profiles.

## Features

- User Registration & Login
- JWT Authentication
- Product/Advertisement Posting
- Image Upload Support
- Product Browsing
- User Profile Management
- SQLite Database Integration
- Responsive React Frontend

## Tech Stack

### Frontend
- React
- React Router
- Vite

### Backend
- Node.js
- Express.js
- SQLite3
- JWT Authentication
- bcrypt
- Multer
